Rivet Tcl packagы

Кроме всего прочего, Rivet - это еще и набор полезных packageй.

  • commserver - это package который может быть использован при IPC. Страшно экспериментальный. Ему нужен comm из tcllib
  • dio - абстрактный интерфейс работы с базами данных.
  • dtcl - модуль для обеспечения совместимости с mod_dtcl
  • form - для создания forms.
  • rivet - еще куча полезных процедур.
  • tclrivet
